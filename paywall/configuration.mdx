### Overview

Every paywall instance can be customized to reflect your monetization strategy, model access, error handling, and UX behavior.

### Settings You Can Configure

- **Pricing per model**: Define request and token costs.
- **Allowed models**: Restrict to a subset of models for each paywall.
- **Free trial balance**: Seed new users with initial credits (soon).
- **Connection behavior**: Customize when users are required to connect (soon).
- **Paywall messages**: Localize or customize paywall responses.
- **Post-payment redirects**: Define where users land after top-up or connection (soon).

All settings are accessible via the [Paywalls.ai](http://Paywalls.ai) dashboard.
